## robe-skeleton-ui

##### Getting Started

##### Download and Install 

```ssh
git clone https://github.com/robeio/robe-skeleton-ui.git 
cd robe-skeleton-ui
npm install
```

##### Development Mode

* start
```ssh
npm start
```
* open browser http://localhost:8080

##### Test

* full test
```ssh
npm test
```

* test file
```ssh
npm testd App
```

* test and coverage
```ssh
npm testc
```

##### Build
* build
```ssh
npm build
```
